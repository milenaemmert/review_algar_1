Variáveis

O que é uma variável?

Uma variável é um espaço na memória para armazenar algum valor, e esse valor pode ser alterado — ou variado, por isso variável.

Imagine uma carteira vazia. Agora adicionamos R$ 100 dentro dela. A carteira passa a ter um valor de cem reais dentro dela. Se gastarmos R$ 40, ela sofrerá uma variação em seu valor, e então terá um valor de R$ 60.

Declarando uma variável

Em JavaScript podemos declarar variáveis usando var ou let. Aqui nos exemplos usaremos let.

Declarar uma variável significa criá-la, mas não quer dizer que ela contenha algum valor.

let carteira

Declaramos usando var ou let e um nome para essa variável.

Inicializando uma variável

Inicializar uma variável significa atribuir a ela algum valor. Colocamos um sinal de igual e o valor:

let carteira = 100

Gastamos R$ 40, então a variável carteira passa a ter um novo valor de 60 reais:

carteira - 40
carteira passa a ter o valor de 60

Perceba que não usamos novamente o let, pois uma vez já declarada a variável, não é mais necessário usar var ou let.

Para saber mais...

=> Por convenção, utilizamos camelCase para nomear variáveis em JavaScript

=> Variáveis em JS podem armazenar números inteiros, decimais, valores booleanos (true ou false), strings (caracteres alfanuméricos entre aspas), listas (arrays)...

let numeroInteiro = 5482
let numeroDecimalPI = 3.1415
let verdadeiro = true
let endereco = “Rua Brasil, 25”
let animais = [‘burro’, ‘gato’, ‘dragão’]
