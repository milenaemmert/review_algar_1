Bibliotecas

O que é uma biblioteca?

Bibliotecas são uma coleção de recursos agrupados para alguma finalidade. São pacotes de código já prontos e testados por alguém ou por alguma corporação, e que facilitam a vida de desenvolvedores otimizando e automatizando tarefas.

São construídas a partir de alguma linguagem e possuem uma documentação, que funciona como um guia.

p5.js

O p5.js é um exemplo de biblioteca gratuita do JavaScript, criada com o propósito de abordar a programação de uma forma mais criativa e torná-la mais acessível para iniciantes, educadores, entre outros.

A função circle(), que usamos para desenhar a bolinha do Pong, é um exemplo de como podemos otimizar uma tarefa usando uma biblioteca.
